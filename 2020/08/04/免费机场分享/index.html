首先下载ssr客户端

这里提供几个下载地址：

ssr解压可运行

ssr下载完成后解压，然后运行ShadowsocksR-dotnet2.0.exe或者ShadowsocksR-dotnet4.0.exe即可。
ssr windows 客户端的运行需要.net的环境支持，一般来说win7以上的系统都可以直接运行。
如果允许错误，那么需要先安装microsoft.net framework 4.

复制SSR订阅链接到ssr服务器订阅

注册成功后 按照图中指示复制ssr订阅链接

配置ssr

打开ShadowsocksR-dotnet2.0.exe或者ShadowsocksR-dotnet4.0.exe，在桌面右下角，右击ssr客户端图标，打开ssr服务器订阅设置

删掉默认的订阅地址，添加刚刚复制的速鹰666的订阅地址

然后更新ssr服务器订阅
